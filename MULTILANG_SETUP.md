# Multi-Language Setup (English + Urdu)

## Option 1: Polylang (Recommended — Free)

### 1. Install Polylang
1. Go to **Plugins → Add New**
2. Search for **Polylang**
3. Install and Activate

### 2. Configure Languages
1. Go to **Languages → Languages**
2. Add **English** (if not already)
3. Add **Urdu** — Select from the list
4. Set **English** as default language
5. Save

### 3. Enable RTL for Urdu
- Polylang automatically handles RTL for Urdu
- Go to **Languages → String translations** to translate theme strings

### 4. Create Translated Pages
- For each page (Pick & Drop, Ziyarat, etc.), use the **Languages** meta box to add Urdu translation
- Or use **Languages → Translations** to link English/Urdu page pairs

## Urdu Content

For Urdu text, use:
- **Unicode Urdu** (recommended) — e.g. اردو
- **Roman Urdu** — e.g. "Yahan type karein" (can be used without RTL)

Astra and the child theme support RTL when Urdu is selected.
